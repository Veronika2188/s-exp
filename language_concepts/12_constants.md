# Constants

### What are Constants?

Constants are values that cannot be changed after they are defined.

### TypeScript

**Example:**

```typescript
const input = '(+ 1 (* 2 3))';
```

### Scala

**Example:**

```scala
val input = "(+ 1 (* 2 3))"
```

### Swift

**Example:**

```swift
let parser = Parser()
```

### Python

**Example:**

```python
input_string = "(+ 1 (* 2 3))"

### C++

**Example:**

```cpp
const std::string input = "(+ 1 (* 2 3))";
```

### C#

**Example:**

```csharp
const string input = "(+ 1 (* 2 3))";
```

### Dart

**Example:**

```dart
const String input = "(+ 1 (* 2 3))";
```

### Kotlin

**Example:**

```kotlin
val input = "(+ 1 (* 2 3))"
```

### Go

**Example:**

```go
const input string = "(+ 1 (* 2 3))"
```

### Haskell

**Example:**

```haskell
input :: String
input = "(+ 1 (* 2 3))"
```

### OCaml

**Example:**

```ocaml
let input = "(+ 1 (* 2 3))"
```

### Rust

**Example:**

```rust
const INPUT: &str = "(+ 1 (* 2 3))";
```

### Lisp

**Example:**

```lisp
(defconstant +input+ "(+ 1 (* 2 3))")
```

### Clojure

**Example:**

```clojure
(def ^:const input "(+ 1 (* 2 3))")
```

### F#

**Example:**

```fsharp
let input = "(+ 1 (* 2 3))"
